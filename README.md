## How to start

Install project dependencies with **npm** inside each folder (backend & frontend)

Note: you will need a **.env** file inside **backend/** folder with the following variables:
  * MONGODB_URI
  * TEST_MONGODB_URI
  * SECRET
  
### For production:
  * `cd backend` 
  * `npm run build:ui`
  * `npm start`
